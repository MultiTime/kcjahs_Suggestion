### 1. 프론트엔드 디자인 및 구현
*   **구조화 (HTML):** `index.html`을 통해 건의함 서비스의 핵심적인 UI 구조를 설계하였습니다.
*   **스타일링 (CSS):** `style.css` 파일이 프로젝트 코드의 가장 큰 비중(41.7%)을 차지하고 있어, 시각적인 디자인과 사용자 경험(UX)에 많은 노력을 기울였음을 알 수 있습니다.
*   **기능 구현 (JavaScript):** `app.js`를 통해 웹사이트의 동적인 기능을 제어합니다.

*   **커밋 활동:** 총 16회의 커밋(Commits)을 통해 지속적인 기능 개선과 유지보수가 진행되었습니다.

## 🚀 설치 및 실행 방법 (How to Run)
이 프로젝트는 정적 웹사이트(Static Website) 구조를 기반으로 하므로 별도의 백엔드 서버 설정 없이 브라우저에서 바로 실행할 수 있습니다.

2.  프로젝트 폴더로 이동합니다.
3.  `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
